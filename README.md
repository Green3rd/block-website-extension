# Focus Redirect Extension
Stay focused by redirecting distracting websites or paths to shhhnoise.com.

## How to use
1. Open Chrome and navigate to `chrome://extensions/`.
2. Enable **Developer mode** in the top right corner.
3. Click **Load unpacked** and select the `/Users/green3rd/Documents/Git/block-website-extension` directory.
4. Once loaded, click the extension icon (or its options page) to manage blocked websites.
5. You can block an entire domain (e.g., `facebook.com`) or a specific path (e.g., `facebook.com/gaming`).
